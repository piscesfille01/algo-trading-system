## 📋 각 날짜 확인 방법

### 날짜별 추천 종목 보기
```bash
# 2025-10-27 추천 종목 확인
cat output/backtest-picks/2025-10-27.json

# 또는 직접 실행해서 보기
node scripts/pick.js 2025-10-27
```

### 진입/패스 입력
```bash
# 예시: 2025-10-27 NVDA 진입 후 성공
node scripts/backtest-add.js 2025-10-27 NVDA entered 118.50 124.00 115.00 hit 123.80

# 예시: 2025-10-27 AMD 패스
node scripts/backtest-add.js 2025-10-27 AMD passed "score only 6"
```

## 🎯 기대 결과

### 상승장 (10개)
- 예상 승률: 70-80%
- 다이버전스 후 반등 확률 높음
- 목표가 도달 가능성 높음

### 하락장 (10개)
- 예상 승률: 40-50%
- 다이버전스가 있어도 실패 가능
- 섹터 약세 패턴 많이 발견 예상

### 횡보장 (10개)
- 예상 승률: 55-65%
- 변동성 작아 목표가 도달 어려움
- R:R 비율 낮을 가능성

### 전체 평균
- 예상 승률: 60-65%
- 시장 환경별 차이 확인
- 개선점 자동 발견

## 📊 분석 시 확인할 것

### 30건 완료 후 분석:
```bash
node scripts/backtest-analyze.js
```

**자동으로 발견할 것:**
1. 상승장 vs 하락장 승률 차이
2. 하락장에서 실패 패턴
3. 횡보장 진입 시 주의점
4. 시장 환경별 점수 조정 필요성

**예상 제안:**
- "하락장 시 최소 점수 10점으로 상향"
- "횡보장 시 R:R 2.0:1 이상만"
- "상승장 시 공격적 진입 OK"

## ⚠️ 주의사항

1. **순서대로 하지 않아도 됨**
   - 편한 날짜부터 해도 OK
   - 상승/하락/횡보 섞어서 해도 OK

2. **모든 날짜 필수 아님**
   - 최소 20개만 해도 분석 가능
   - 각 시장 환경당 최소 5개씩만

3. **정확한 데이터 입력**
   - 실제 진입가/청산가 입력
   - 당일 추천가와 다를 수 있음 (갭 등)

4. **임시 파일 정리**
   ```bash
   # 스캔 완료 후
   rm output/temp-pick-*.txt
   ```
